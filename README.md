# Medico - Clínica de Fisioterapia

Proyecto desarrollado con React, TypeScript y Tailwind CSS.

## Requisitos previos

- Node.js (v18 o superior recomendado)

## Instalación

1. Clona el repositorio.
2. Instala las dependencias:

```bash
npm install
```

3. Crea un archivo `.env.local` si necesitas configurar variables de entorno (como claves de API).

## Ejecutar localmente

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```

## Construcción (Build)

Para generar la versión de producción optimizada:

```bash
npm run build
```

## Estructura del Proyecto

- `/components`: Componentes reutilizables (Header, Footer, etc.).
- `/pages`: Vistas principales de la aplicación.
- `/data`: Datos estáticos (listas de servicios, precios, enlaces).
- `/constants`: Constantes globales como rutas de imágenes.
